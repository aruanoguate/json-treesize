# Changelog

All notable changes to the **XML Tree Size** extension will be documented in this file.

## [0.1.0] — 2026-04-10

### Added

- Interactive tree explorer for XML and SOAP payloads
- Detail bar chart pane with size-ranked child breakdowns
- Support for XML elements, attributes, text nodes, and CDATA sections
- Theme-aware heat-map coloring with configurable base color
- Right-click Explorer integration for `.xml` files
- Command Palette support: **XML Tree Size: Analyze**
- One-click navigation from nodes back to the source XML
- Automated demo recording with Playwright (command palette + interaction GIFs + screenshot)
- README media assets and XML-specific demo fixture
- 20-language runtime localization bundle support

### Changed

- XML package now follows the same monorepo demo and release workflow shape as JSON Tree Size
